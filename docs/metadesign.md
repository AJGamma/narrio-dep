# 开发者prompt调优管线

此管线的目的是方便开发者对assets资产进行快速迭代。以下是详细的内容设计。

## 多变体并行运行

当前开发的一大痛点是，基于CLI的测试交互过于繁琐、每次只能运行一个变体，且运行时的中间过程不可见。为了解决这一问题，需要一个多变体并行运行的调优管线。具体地：

用户可以输入一个文件路径和多个风格，通过CLI并行运行，并在此过程中看到每个变体的中间结果。这样可以大大提高开发效率，减少等待时间。

技术栈：基于tmux、tmuxifier以及kitty。tmux和tmuxifier用于窗口多开、窗口管理，kitty用于在终端中展示最终
