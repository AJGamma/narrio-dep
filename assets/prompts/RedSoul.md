# 小红书图文哲学
## 强烈的「杂刊」排版风格（Editorial Layout）
- 小红书的视觉语言非常依赖于非对称的、具有呼吸感的排版，模仿现代独立杂志的审美：
- 留白与错落感： 标题、正文和插图并不总是居中对齐，而是通过错落的排布创造节奏感。
- 标题装饰化： 标题可能带有材质、框线或手写体修饰，增强视觉重点。
- 模块UI化信息流： 将复杂的信息（竞赛流程/技术原理/时间轴）拆解成一个个独立的小卡片或步骤，实现信息 UI 化。
  
## 「极简」与「极繁」的动态平衡
- 核心元素突出： 每一张海报都有一个绝对的视觉中心，确保用户在快速滑动时能瞬间捕捉主题。
- 细节丰富度： 在中心元素周围散布着细小的辅助元素，这种“精密感”提升了设计的专业度。
- 强一致性：每一套图的风格/色彩/材质/构成是完全统一的
